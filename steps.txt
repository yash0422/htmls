STEPS TO HOST PAGES UNDER FOLDERS:

1. You'll land on the repo homepage — click Add file → Upload files
2. Create your folder structure by typing the path directly in the filename field — GitHub accepts it:
Type htmls/oil-chart/index.html and it auto-creates the folders
3. Drag and drop your HTML file
4. Click Commit changes
5. Go to Settings → Pages → Branch → main → Save
6. Wait ~2 minutes, then your URL is live: https://yourusername.github.io/htmls/oil-chart/
